Soy un entusiasta de la enseñanza, orientación y ayuda hacia mis compañeros que lo necesitan.

Habilidades

Frontend

El frontend es con lo que el usuario interactúa en las aplicaciones web.

Backend

El backend es la lógica que hay detrás de una aplicación web.

Base de datos

Es el donde se almacena la información de las aplicaciones tanto web como nativas de un Sistema Operativo

Aplicaciones nativas

Son las aplicaciones que se desarrollan exclusivamente para un Sistema Operativo, por lo que está más homogeneo con el sistema.

DevOps

Es el proceso automático que tiene un proyecto para diseñar, desarrollar, construir, probar y desplegar la aplicación en un servidor en la nube.

UI/UX

Son las buenas prácticas que toda aplicación debe tener para que el usuario se sienta cómodo con la aplicación
